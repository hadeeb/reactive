# Reactive

`Reactive` is yet another global state manager for react apps

- ~1 KB (min+gzip)
- Automatic dependency tracking with ES6 Proxies.
- Mutate the state to update the UI.
- First class typescript support.

## Install

```
yarn add @hadeeb/reactive

// npm install @hadeeb/reactive
```
